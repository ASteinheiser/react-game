# RPG made with React + Redux
Turn-based, Dungeon-crawling RPG made with the finest web technologies! [Play the latest version here!](http://react-rpg.com)

# Get Started Locally!
## Install
```
git clone https://github.com/ASteinheiser/react-rpg.com
cd react-rpg.com
yarn install
```
## Develop Locally
```
yarn start
```
## Run Tests
```
yarn test
```
